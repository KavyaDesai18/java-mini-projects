# Java Mini Projects

This repository contains beginner Java console applications built to practice programming fundamentals.

Projects Included:
- Alarm Clock
- Audio Player
- Banking Program
- Calculator
- Countdown Timer
- Dice Rolling Game
- Guess Game
- Hangman Game
- Quiz Game
- Rock Paper Scissors Game
- Slot Machine Game

Technologies Used:
- Java
- Object-Oriented Programming
- Console Input/Output
